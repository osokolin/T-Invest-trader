# Workflow: next-step

1. Planner proposes one small milestone.
2. Human approves or trims scope.
3. Architect checks architecture fit before implementation.
4. Implementer applies the change.
5. Tester runs the smallest appropriate verification.
6. Reviewer reviews the delta.
7. If green, Committer commits.
8. If not green, run `workflows/fix-pass.md`.
